## 推广文本 (170字以内)

## 描述

Peacock 是一款快速、私密的跨平台局域网文件与消息传输工具，所有数据完全在本地网络中传输。

零配置
在同一 Wi-Fi 网络下的设备上打开 Peacock，设备自动发现彼此——无需注册账号、无需配对、无需扫码。

即时消息
设备间实时发送文字消息，消息仅在本地网络传输，不经过任何服务器。

文件传输
直接发送照片、文件和文件夹，以局域网满速传输，支持进度显示和断点续传。

片段与快速复制
创建和管理文本片段——代码片段、API Key、笔记或任何可复用的文本。标记任意文本即可一键复制。一键分享片段到任何设备。

隐私至上
- 所有数据仅在本地网络传输
- 无云端服务器、无账号、无追踪
- 无需联网即可使用
- 开源项目（MIT 协议）

功能特性
- 自动设备发现（UDP 广播）
- 实时文字消息
- 照片、文件和文件夹传输
- 断点续传
- 文本片段管理与快速复制
- 双语界面（中文 / English）
- 暗色主题
- 支持 Windows 和 Linux 电脑互传

使用原生 Swift 和 SwiftUI 构建，带来最佳 iOS 体验。
